# Planned Roadmap

#### Jun 30 2023 - v1.0.1 released
- Main repository updated to use a BSD-3 license, with limited exceptions (link?)
- Line credits now shown for all authors
- We are actively accepting PR's and feature requests! Please submit requests via the [REQUEST] git issue template
- <b> If possible, please clone a new copy of the repo when you update </b>

#### Mid July
- Code release for water, fire and smoke simulations shown in the intro video
- Code release for improved tree branches and fruit placement shown in intro video
- Code release for two additional plant species.
- Scene reproducibility improvements
- Disk usage and system stability improvements
- New tutorial on how to generate specific assets rather than full scenes

#### End of July
- Initial pre-generated data release. We will continuously render and release new data from this point onwards.
- New tutorial on how to create your own procedural assets
