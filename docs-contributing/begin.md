# Contributing to Infinigen

We welcome contributions! You can contribute in many ways:
- **Contribute code** - We welcome code contributions to the [Infinigen repo](https://github.com/princeton-vl/infinigen). More guidelines coming soon.
- **Contribute procedural generators** - Infinigen provides tools to convert artist-friendly [Blender Nodes](https://docs.blender.org/manual/en/2.79/render/blender_render/materials/nodes/introduction.html) into python code. Tutorials and guidelines coming soon.
- **Contribute pre-generated data** - Anyone can contribute their computing power to create data and share it with the community. Please stay tuned for a repository of pre-generated data.
