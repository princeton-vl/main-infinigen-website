Infinigen covers 14 different scene types, each with their own gin configuration options:
1. Arctic
2. Canyon
3. Cave
4. Cliff
5. Coast
6. Coral Reef
7. Desert
8. Forest
9. Kelp Forest
10. Mountain
11. Plain
12. River
13. Snowy Mountain
14. Under Water

### Running Single Scenes with Specific Types

For single scenes, scene types are specified with the `--gin_config` or `-g` flag. 
For instance, we can generate a coarse terrain for a river scene with:
```
../blender/blender --background -noaudio --python generate.py -- --seed 0 --task coarse --output_folder outputs/dev/coarse --gin_config river
```
We can then generate a full river scene with populated assets and fine terrain: 

```
../blender/blender --background -noaudio --python generate.py -- --seed 0 --task populate fine_terrain --input_folder outputs/dev/coarse --output_folder outputs/dev/fine --gin_config river
```

### Running Pipelines with Specific Scene Types
The scene types can also be specified in for a full pipeline using `--config`.
For instance, we can render a snowy mountain scene with:

```
python -m tools.manage_datagen_jobs -o outputs --config snowy_mountain
```
